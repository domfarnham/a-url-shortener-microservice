A URL Shortener Microservice By dom.farnham('@JSCodebook')

A microservice which takes a URL, passed as a parameter to the API, and then responds with a shortened URL in JSON.

Example usage:

https://shorten-my-url.glitch.me/api/https://www.google.com

Example output:

{"originalURL":"https://www.google.com", "shortURL":"https://shorten-my-url.glitch.me/1000"}

Demo:

https://shorten-my-url.glitch.me/1000 redirects to https://www.google.com
