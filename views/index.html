A URL Shortener Microservice By dom.farnham()

A microservice which takes a URL, passed as a parameter, and then responds with a shortened URL in JSON.

Example usage:

https://shorten-my-url.glitch.me/api/https://www.google.com

Example output:

{"originalURL":"https://www.google.com", "shortURL":"https://shorten-my-url.glitch.me/8170"}

Demo:

https://shorten-my-url.glitch.me/8170 redirects to https://www.google.com
